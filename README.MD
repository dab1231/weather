Weather Viewer

Приложение для просмотра погоды в различных городах. Использует API для получения данных о погоде и отображает их в удобном интерфейсе.

Стек технологий:

- Java 17
- PostgreSQL
- Spring MVC
- Flyway
- Hibernate 
- Apache Tomcat
- OpenWeatherMap API
- Bootstrap 5
- Thymeleaf

Как запустить локально:

1. Клонируйте репозиторий на свой компьютер.
2. Установите PostgreSQL и создайте базу данных для приложения.
3. Установите необходимые зависимости, используя Maven.
4. Настройте файл `application.properties` для подключения к вашей базе данных.
5. Соберите проект командой mvn clean package 
6. Запустите проект через Tomcat.
